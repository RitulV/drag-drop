## Features

- Kanban-style workflow:
  - **To Do**
  - **In Progress**
  - **Completed**
  - **Under Review**
- Create todos with:
  - Title & description
  - Priority (Low, Medium, Urgent)
  - Due date
- Color-coded priority badges
- Add, update, and delete tasks
- Clean, minimal UI for easy task tracking

- **Frontend:** React + TypeScript
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
